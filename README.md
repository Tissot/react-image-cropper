# react-image-cropper

基于 HTML5 - Canvas 实现的 React 图片裁剪组件。

## Demo

暂时还比较简陋，demo 及组件源码在 demo 分支。

URL：https://tissot.github.io/react-image-cropper/

## Features

无需把图片及其裁剪坐标、高宽传到后端裁剪，基于 HTML5 - Canvas 实现了在前端对图片进行裁剪的功能。

## Usage

暂时只支持把 demo 分支下载下来然后 import 组件。
